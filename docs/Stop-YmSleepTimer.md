# Stop-YmSleepTimer

## SYNOPSIS
The function turns off  the sleep timer.

## SYNTAX

```
Stop-YmSleepTimer [-DeviceAddress] <String[]> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The function turns off  the sleep timer.
The function sets sleep timer to 0.

## EXAMPLES

### EXAMPLE 1
```
Stop-YmSleepTimer -DeviceAddress 10.10.0.30
```
